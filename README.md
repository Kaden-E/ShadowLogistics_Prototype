# Shadow Logistics

Shadow Logistics is a systemic strategy prototype built in Unity (2D URP).

The game explores political risk management through cross-border logistics in a fictional three-country region.

## Current Prototype State

- Vertical 3-country geopolitical layout
- 6 town nodes
- Motorway and backroad border crossings
- Basic route connection structure (Country A complete)

## Design Pillars

- No pure bad-luck failures
- Risk driven by player decisions
- Escalating visibility and permanent reputation floors
- Opaque causality, visible consequences
- Political risk > gambling

## Tech

- Unity 2D URP
- LineRenderer for route connections
- Clean hierarchical structure for future system expansion
